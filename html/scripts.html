RoundEdges module¶

Contains multiple reusable tools for the BladeGen module.

class scripts.roundedges.RoundEdges(edge, x, y, r, camberline)[source]¶

Bases: object

Class for fitting in leading edge and/or trailing edge radius. Either the front half (for LE fitting) or the latter half (for TE fitting) will be taken from input. A quarter circle will be attached to upper and lower surfaces, rotated into position for a smooth transition. The intersection between the quarter circles will be determined and cut, emerged gap will be filled with interpolation.

Returns the x,y coordinates of original length with the fitted radius.

Parameters

  • edge (str) – TE/LE

  • x (ndarray) – x coordinates of whole blade

  • y (ndarray) – y coordinates of whole blade

  • r (float) – radius of LE/TE circle

  • camberline (list of ndarrays) – xy of camberline

Returns

x, y
