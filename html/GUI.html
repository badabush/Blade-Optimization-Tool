Main window¶

The Graphical User Interface for the BladeGen script.

class scripts.UI.gui.PlotCanvas(parent=None, width=5, height=4, dpi=100)[source]¶

Bases: matplotlib.backends.backend_qt5agg.FigureCanvasQTAgg

All the plotting commands are organized here. At GUI start, an empty empty figure is generated. On Update, BladeGen will be called with the user parameter and the plot will be updated.

class scripts.UI.gui.Ui[source]¶

Bases: PyQt5.QtWidgets.QMainWindow, UI.update_handle.UpdateHandler

Load UI from .ui file (QT Designer). Load restraints for parameters (min, max, default, step) from restraints.txt. Parameters with values or steps <1 have to be scaled since the slider only accepts int values. Slider and Textboxes are linked, so change one will modify the other (and vice versa). Plot will save zoom state even on updating plot.
