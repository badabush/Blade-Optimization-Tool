assert_input(nblade, r_th, alpha1, alpha2, x_maxcamber, rth_le, rth_te, npts)[source]¶

Assert input parameter are within defined range.

Parameters

  • nblade (str) – number of blades, default: ‘single’

  • r_th (float) – relative thickness, default: .0215

  • alpha1 (float) – inlet angle, default: 25

  • alpha2 (float) – outlet angle, default: 25

  • x_maxcamber (float) – X position of max. chamber, default: .4

  • rth_le (float) – rel. thickness leading edge as percentage of chord, default: .02

  • rth_te (float) – rel. thickness trailing edge as percentage of chord, default: .0135

  • npts (int) – number of blade points, default: 400
